# 一、CGHub全局介绍

## 1.1 创客星球是什么？

**创客星球(CGHub)**（Creators Galaxy Hub）是基于"AI+Web3+共同富裕理念"的创新实践平台，探索人类文明的未来形态。

**愿景：** 和合共生、天下大同
**使命：** 创新实践人类命运共同体蓝图，为人类文明的未来新形态探路
**Slogan：** 点燃创客之光，点亮未来梦想

## 1.2 核心公式

```
AI生产力 × Web3生产关系 × 共同富裕理念 = 创客星球(CGHub)
```

- **AI生产力** — 一个人就是一支军队
- **Web3生产关系** — 规则即代码、创造即确权、贡献即回报
- **共同富裕** — 不是零和博弈，是共赢生态

## 1.3 为什么需要创客星球？

AI解决的是"做事"的问题，但解决不了"成事"的问题。

一个人+AI可以完成工作，但一个人无法形成：
- **传播网络** — 裂变、口碑、网络效应
- **信任传递** — 真实人际背书
- **价值分配** — 新增价值被谁拿走

```
AI = 做事工具
创客星球(CGHub) = 成事组织

一个人 + AI = 能做
一个人 + AI + 创客星球(CGHub) = 能做大
```

## 1.4 长期愿景

创客星球是开源开放自由自治的去中心化社区，
是个体价值创造与分配生态操作系统，
是AI生产力跃迁驱动生成关系变革背景下共同富裕探路者的自组织实践，
是对人类未来新组织形态和社会文明的探索。

创客星球是有生命力并不断涌现自生长的生态型自组织，不依赖于融资输血，而是从一开始就要尝试自我造血打造可运营的商业闭环，鼓励参与者积极创造价值并获得公平合理的回报收益，形成可循环迭代指数型裂变增长的生态系统。

# 二、MVP黑客松目标

## 2.1 Demo演示核心

**贡献记录 → x402证明 → Cobo Agentic Wallet自动收益分配**

这是 **A2A Economy / Agent-Native Payments** 的真实演示：
**AI Agent 自己持有钱包、自己发起分配。**

不要求功能完整，不要求UI美观——**要求闭环跑通，能录屏/截图证明**。

## 2.2 技术架构

```
AI Agent（贡献者）
    ↓ 贡献行为触发
贡献记录合约（链上）
    ↓ 生成证明
x402协议（证明标准）
    ↓ 验证通过
Cobo Agentic Wallet（自动收益分配）
    ↓
收益到达Agent钱包
```

## 2.3 技术栈

| 模块 | 技术 |
|------|------|
| 链侧 | Hardhat + Polygon Mumbai（或Monad Testnet）|
| 合约 | Solidity（ContributionLedger + 分配合约）|
| 前端 | Next.js + TypeScript + wagmi |
| 钱包 | Cobo Agentic Wallet SDK |
| 证明 | x402协议 |
| Agent | 贡献记录 + 证明生成 + 调度逻辑 |
